# EchoLearn Conversion Modes

EchoLearn supports two future learning experiences: Audiobook Mode and
EchoLesson Mode.

## Audiobook Mode

Audiobook Mode converts documents into continuous listening audio. It is the
default mode and keeps the current EchoLearn behavior: automatic language
detection, the selected English and Spanish voices, and minimal interruptions.
Auto Learning Pauses are off by default so the experience stays focused on
listening.

Use Audiobook Mode for:

- Books
- Articles
- CCNA study material
- Technical documentation
- General PDFs that should sound like a continuous audiobook

## EchoLesson Mode

EchoLesson Mode is for educational content that should eventually become
structured learning audio instead of a continuous read-through.

Future versions will use an AI Lesson Builder to transform PDFs into lesson
markup with sections such as:

- `[TITLE]`
- `[FLOW]`
- `[EXPLANATION]`
- `[DIALOG]`
- `[SPEAKER_1]`
- `[SPEAKER_2]`
- `[PRACTICE]`
- `[REVIEW]`
- `[PAUSE_X]`

Use EchoLesson Mode for future workflows such as:

- Language learning
- Courses
- Dialogues
- Practice exercises
- AI-built lessons from PDFs

EchoLesson Mode is currently an application architecture and UI foundation. It
does not change audio generation yet.
